# Time control

## Usage

```
npm install @eox/timecontrol-2
```

```
// for setups with bundlers e.g. Vite
import "@eox/timecontrol-2"

// or, for e.g. single HTML files without bundler
import "@eox/timecontrol-2/dist/eox-timecontrol-2.js"

<eox-timecontrol-2></eox-timecontrol-2>
```

Please refer to the [timecontrol-2 docs](https://eox-a.github.io/EOxElements/?path=/docs/elements-eox-timecontrol-2--docs) for more details.
